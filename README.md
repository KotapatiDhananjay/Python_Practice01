# Python Practice 01

This repository contains simple Python programs for beginners to practice basic programming concepts.

## Programs Included

### 1. Basic Calculator

This program performs the following arithmetic operations on two numbers entered by the user:

* Addition
* Subtraction
* Multiplication
* Division

#### Sample Output

```text
Enter the first number: 10
Enter the second number: 5

Addition is 15.0
Subtraction is 5.0
Multiplication is 50.0
Division is 2.0
```

---

### 2. Greeting Program

This program accepts the user's first name and last name and displays a welcome message.

#### Sample Output

```text
Enter your first name: John
Enter your last name: Doe

Hello, John Doe Welcome to python program
```

---

## Requirements

* Python 3.x

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/KotapatiDhananjay/Python_Practice01.git
```

2. Navigate to the project directory:

```bash
cd Python_Practice01
```

3. Run the Python program:

```bash
python filename.py
```

Replace `filename.py` with the name of the Python file you want to execute.

## Learning Objectives

* Taking user input
* Using variables
* Performing arithmetic operations
* Printing formatted output
* Working with strings
